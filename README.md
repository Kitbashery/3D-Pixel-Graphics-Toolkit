# 3D Pixel Graphics Toolkit

HDRP shader graphs for creating pixel-style graphics in 3D.

## Features:

* Pixelate Shader
* Pixel Cloud Shader
* Pixel Fluid Shader
* Pixel Triplanar Shader

### Pixel-Perfect Primitives:
Meshes with UVs optimized for pixel rendering.

* Cube
* Cylinder
* Ramp
* Sphere
* Dome
